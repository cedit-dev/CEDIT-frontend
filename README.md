# CEDIT - Centro de Desarrollo e Innovación Tecnológica 🚀

![UFPSO Banner](https://raw.githubusercontent.com/lucide-react/lucide/main/icons/layers.svg) <!-- Reemplazar con banner real si existe -->

Este es el repositorio oficial del frontend del **CEDIT**, la unidad de innovación tecnológica de la **Universidad Francisco de Paula Santander Seccional Ocaña**. El proyecto está diseñado para impulsar la transformación digital en la región del Catatumbo, conectando la academia con el sector productivo.

---

## ✨ Características Principales

- **Diseño Premium:** Una interfaz moderna, limpia y profesional con estética *Glassmorphism*.
- **Responsive Design:** Optimizado para dispositivos móviles, tablets y desktops.
- **Animaciones Suaves:** Transiciones dinámicas al hacer scroll y micro-interacciones interactivas.
- **Secciones Clave:**
  - **Hero Section:** Introducción impactante con orbes dinámicos.
  - **Sobre el CEDIT:** Misión, visión e historia del centro.
  - **Servicios Tech:** Catálogo de servicios de fomento tecnológico y apoyo empresarial.
  - **Portafolio de Proyectos:** Galería filtrable de proyectos en Realidad Virtual, Web y Capacitación.
  - **Programas:** Detalle de iniciativas actuales como Punto Vive Lab y Tiendas Digitales.
  - **Contacto:** Formulario integrado y geolocalización.

---

## 🛠️ Stack Tecnológico

El proyecto utiliza las tecnologías más modernas para garantizar rendimiento y escalabilidad:

- **Core:** [React](https://reactjs.org/) + [TypeScript](https://www.typescriptlang.org/)
- **Build Tool:** [Vite](https://vitejs.dev/)
- **Estilos:** [Tailwind CSS](https://tailwindcss.com/)
- **Iconografía:** [Lucide React](https://lucide.dev/)
- **UI Components:** [Radix UI](https://www.radix-ui.com/) + [Shadcn/UI](https://ui.shadcn.com/)
- **Hook de Visibilidad:** Custom `useInView` para animaciones basadas en scroll.

## 🚀 Instalación y Desarrollo

Sigue estos pasos para ejecutar el proyecto localmente:

### 1. Clonar el repositorio
```bash
git clone https://github.com/tu-usuario/CEDIT-frontend.git
cd CEDIT-frontend
```

### 2. Instalar dependencias
Puedes usar `npm` o `bun`:
```bash
npm install
# o
bun install
```

### 3. Ejecutar en modo desarrollo
```bash
npm run dev
# o
bun dev
```

El proyecto estará disponible en `http://localhost:5173`.
